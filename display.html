Common questions

Display & wiring FAQ

My scooter has an error code on the display — what does it mean?
Error codes vary by brand and indicate different faults — battery, motor, controller or communication issues. Bring it in or send us a photo of the code via WhatsApp and we'll tell you what we think it means before booking you in.
My scooter got rained on — should I worry?
Most modern e-scooters have some level of water resistance, but connectors and seals degrade over time. If it's still working fine after light rain, you're probably okay. If it's been through standing water, stopped working after rain, or you're seeing corrosion, get it checked sooner rather than later — water damage gets worse the longer it's left.
I don't know what's wrong, it just stopped working — can you still help?
Yes, this is exactly what our fault finding service is for. We test through the whole electrical system methodically until we find the cause, rather than guessing and swapping parts.
My throttle feels sticky or doesn't return properly — is that dangerous?
Yes, this should be treated as a safety issue and not ridden until fixed, since a stuck throttle can cause unintended acceleration. Book it in as soon as possible.
Do you need an appointment?
Yes — appointment only. Call or WhatsApp to book. For unknown faults it helps to describe exactly what happens (when it cuts out, any error codes, recent weather exposure) so we can prepare.
